# DSA 210 Project

This repository contains my DSA 210 term project work on house price analysis and prediction.

## Project Topic
The project focuses on analyzing the determinants of house prices and building prediction models using housing-related features.

## Current Progress
This stage of the project includes:
- data collection from Kaggle
- data cleaning and preprocessing
- exploratory data analysis (EDA)
- hypothesis testing on important housing variables

## Dataset(s)
The project uses the following Kaggle datasets:
- House Prices - Advanced Regression Techniques
- House Price Prediction Dataset

## Main Notebook
The main notebook for the April 14 submission is:

`notebooks/eda_hypothesis_tests.ipynb`

## Variables Examined
Some of the main variables analyzed in this stage are:
- SalePrice
- GrLivArea
- OverallQual
- GarageCars
- Neighborhood
- HouseAge
- TotalSF
- HasGarage

## Notes
Although two housing datasets were collected for the project, the main analysis in this stage focuses on the House Prices - Advanced Regression Techniques dataset because it provides a richer set of explanatory variables for EDA and hypothesis testing.
